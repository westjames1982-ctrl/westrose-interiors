# Westrose Interiors - Modern Commercial Interior Website

A premium, bold, and modern Next.js website for Westrose Interiors, a commercial interior systems company based in Calgary, Alberta.

## Features

- **Hero Section**: Bold, modern introduction with CTA buttons
- **Portfolio Gallery**: Showcase of 23+ completed commercial projects with lightbox modal
- **Services Section**: Comprehensive list of offered services with icons
- **Why Choose Us**: Key differentiators including safety, architect-friendly approach, on-time delivery, and strong relationships
- **Contact Form**: Professional quote request form with form validation
- **Responsive Design**: Mobile-first, fully responsive layout
- **Modern Design**: Dark/light color scheme with teal accent colors
- **Professional Footer**: Company information and quick navigation

## Tech Stack

- **Framework**: Next.js 14
- **Styling**: Tailwind CSS 3
- **Language**: TypeScript
- **Images**: Next.js Image optimization

## Installation & Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
